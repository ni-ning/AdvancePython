===============================
1.1 课程介绍
===============================

----------
你是否遇到
----------
- 看不懂优秀库和框架的源码
- 不知道如何才能进一步优化自己的代码
- asyncio、tornado 等异步框架背后的原理
- python 代码灵活背后的设计原理
- 对生成器稀里糊涂
- 很多 python 抛的异常看不懂


----------
如何进阶
----------
- 和大神一起
- 阅读优秀源码、懂原理
- 面试、多做项目


----------
课程收获
----------
- 应对 python 高级工程师面试
- 从深度和广度都扩充自己的知识体系
- 能应对各种难度的框架和库的源码
- 深入理解 python 的设计理念和背后的原理
